# Instagram Automation Platform for Online Stores

This platform uses the Instagram Graph API to help businesses sell products and manage customer conversations automatically.

### Main Features

* **Comment to DM**: When someone comments on a post, the system automatically sends them a DM.
* **DM Automation**: Automatically reply to customer messages.
* **Story Reply to DM**: When someone replies to a story, the system can start a DM conversation.
* **Flow Builder**: Create automation workflows with an easy visual builder.
* **Customer Tracking**: Save all interactions (comments, DMs, story replies, product inquiries) under each Instagram user.

### Product Management

Store owners can create products manually or convert Instagram posts, reels, or images into products.

For each product, the seller can add:

* Product name
* Price
* Description
* Purchase link
* Other details

### Product Inquiry in DMs

When a customer asks about a product through:

* Product name
* Product ID
* Hashtag
* Shared Instagram post/reel

The system automatically finds the product and links it to the conversation.

### Selling Process

* Customer asks about a product in DM.
* The system identifies the product.
* The seller can answer questions or negotiate the price.
* The seller can send a purchase link directly from the platform.
* The seller can adjust the price before sending the purchase link if needed.
* The customer completes the purchase using the link.

### Benefits

* Faster customer responses
* Better engagement tracking
* Easier product sales through Instagram DMs
* Complete customer history in one place
* More conversions and sales
